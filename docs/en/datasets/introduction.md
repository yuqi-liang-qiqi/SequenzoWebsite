# Datasets

Sequenzo includes example datasets for learning sequence analysis workflows and for checking whether code runs as expected. Use this page to choose a dataset that matches the type of sequence question you want to practice.

## Choose a Dataset

| If you want to practice... | Start with |
| --- | --- |
| A compact social sequence workflow | [MVAD School-to-Work Transitions](/en/datasets/mvad) |
| Family life-course trajectories | [Biofam Family Life Trajectories](/en/datasets/biofam) |
| Multidomain family trajectories | [Biofam Three-Domain Family Life](/en/datasets/biofam-three-domains) |
| Pairfam family or activity trajectories | [Pairfam Family Trajectories](/en/datasets/pairfam-family), [Pairfam Activity Trajectories](/en/datasets/pairfam-activity) |
| Polyadic or intergenerational sequence analysis | [Intergenerational Family Data for Polyadic Analysis](/en/datasets/dyadic-sequence-data-lsog) |
| Historical or comparative examples | [CO2 Emissions](/en/datasets/CO2-emissions), [GDP per Capita](/en/datasets/gdp-per-capita), [Political Science Aid Datasets](/en/datasets/political-science-aid-datasets) |
| Stress and repeated-measure data | [College Students' Stress](/en/datasets/stress-data) |

## Typical Use

Most dataset pages show the source, the variable structure, and the columns that can be passed to [`SequenceData`](/en/function-library/sequence-data). After choosing a dataset, continue with the [Quickstart](/en/basics/quickstart) or the [Typical Workflow](/en/basics/typical-workflow).
