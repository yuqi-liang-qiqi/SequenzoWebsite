# 如何引用我们的 Sequenzo 和社会序列分析的文档网站

Sequenzo 生态系统由两个密切相关但根本不同的组件组成：

**1. Sequenzo 包**

Sequenzo Python 包是一个高性能工具包，用于可扩展且易于访问的社会序列分析。

它提供高效的数据结构、相异性度量、聚类工具、可视化工具和计算增强功能，专为现代大规模序列数据集而设计。

**2. 文档网站**

文档网站不仅仅是包的说明手册，它还包含广泛的教程、概念解释和实践演练。

这些材料不仅展示了如何使用 Sequenzo，还介绍了社会序列分析中的关键思想、方法和工作流程。

我们的目标是支持学习者和研究人员，从初学者到高级用户，理解序列分析的理论和应用实践。

因此，我们将这个官网称之为“Sequenzo 和社会序列分析的文档网站”。

## 如何引用

为确保正确归属，请使用以下引用指南：

**如果您在研究中使用了 Sequenzo Python 包**

请引用 Sequenzo 包论文（即将发布）。

**如果您参考、引用或依赖文档网站中的材料（包括教程、概念解释或方法说明）**

请引用文档网站的 Zenodo 记录：

Liang, Y., Li, X., & Meyerhoff-Liang, J. H. E. (2025).
*Documentation Website for Sequenzo and Social Sequence Analysis.* Zenodo.
[https://doi.org/10.5281/zenodo.17698151](https://doi.org/10.5281/zenodo.17698151)
